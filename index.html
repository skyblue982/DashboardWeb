GitHub deployment

แนวทางขึ้นระบบจริง

  1. เก็บ frontend ไว้ใน GitHub repository เดียว
  2. ใช้ Supabase เป็นฐานข้อมูลและ auth สำหรับผู้ใช้งาน
  3. Deploy หน้าเว็บบน Vercel หรือ Netlify
  4. เชื่อม API ผ่าน Supabase JS เพื่อบันทึกข้อมูลจริง
  5. ตั้ง cron job หรือ edge function สำหรับสรุปรายเดือนถ้าต้องการ
