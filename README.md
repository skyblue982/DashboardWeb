# PDSRCA Property Dashboard

ต้นแบบเว็บสำหรับแปลงข้อมูลจากไฟล์ `ค่าเช่าพี่ปรีดา.xlsm` ให้เป็นระบบติดตามกำไร-ขาดทุนการลงทุนอสังหาแบบ interactive dashboard โดยตั้งใจให้พร้อมต่อยอดขึ้น GitHub และเชื่อม Supabase ได้ทันที

## สิ่งที่มีในโปรเจกต์

- `index.html` หน้า dashboard prototype
- `styles.css` งานออกแบบ UI แบบ responsive
- `app.js` ข้อมูลตัวอย่างและ interaction ของหน้าเว็บ
- `supabase-schema.sql` โครงสร้างฐานข้อมูลสำหรับใช้งานจริง
- `tools/extract_excel.py` สคริปต์ดึงข้อมูลจาก Excel ไปเป็น JSON สำหรับ seed

## แนวทางใช้งาน

1. เปิด `index.html` เพื่อดูต้นแบบหน้าตาและ flow การใช้งาน
2. รัน `python tools/extract_excel.py` โดยตั้ง `PYTHONPATH=./.vendor` ถ้าต้องการ export ข้อมูลจาก Excel
3. นำ `supabase-schema.sql` ไปสร้างตารางใน Supabase
4. เปลี่ยนข้อมูล mock ใน `app.js` เป็นการอ่านจาก Supabase client
5. Deploy ขึ้น GitHub แล้วต่อไปยัง Vercel หรือ Netlify

## แนวคิดหน้าเว็บ

- หน้าแรกเน้น KPI ใหญ่ 4 ตัว: กำไรสุทธิ, ค่าเช่ารวม, ค่าใช้จ่ายรวม, มูลค่าลงทุน
- มีกรองตามเจ้าของห้องและปีเพื่อสลับมุมมองเร็ว
- มีกราฟ ROI รายห้องและ owner split เพื่อเห็นห้องที่ทำผลงานดี
- มีหน้าบันทึกค่าเช่าและค่าใช้จ่ายแยกกันชัดเจน ลดการกรอกผิดประเภท
- รองรับการต่อยอดเป็นระบบจริงที่มี auth, audit trail และรายงานรายเดือน

## ข้อเสนอแนะตอนทำเวอร์ชัน production

- ใช้ Next.js + Supabase JS ถ้าต้องการระบบจริงที่ขยายต่อได้
- เพิ่มหน้า login และสิทธิ์การเข้าถึงข้อมูล
- แยกตาราง `leases` ถ้าต้องการเก็บข้อมูลผู้เช่า สัญญา และวันหมดสัญญา
- เพิ่มหน้า cashflow รายเดือน และ alert สำหรับห้องว่างหรือค่าใช้จ่ายผิดปกติ
